# TimeBinTracker

TimeBinTracker is a minimal computer use tracker I use to track my activity across multiple computers in real time. 

Unlike the [ActiveWindowLogger](https://github.com/swharden/ActiveWindowLogger) project I created last year, TimeBinTracker only records binary activity (not window titles), is suitable for simultaneous use across multiple computers, and uploads activity to a web endpoint to allow real-time visualization of activity via a web page.
